== Description ==

Adds inaccessible AJAX-driven handling of comment submission.

DO NOT USE THIS PLUG-IN! It was created to demo a broken version of https://github.com/joedolson/accessible-ajax-comments

== Installation ==

1. Upload the `inaccessible-ajax-comments` folder to your `/wp-content/plugins/` directory
2. Activate the plugin from the `Plugins` menu in WordPress
